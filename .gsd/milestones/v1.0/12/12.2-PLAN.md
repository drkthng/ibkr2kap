# Plan 12.2: Streamlit Tax Processing & Excel Export

## Objective
Implement tabs in the Streamlit UI to run the FIFO matching engine, calculate taxes (tax pools), and allow the user to download the Anlage KAP Excel report.

## Context
- src/ibkr_tax/services/fifo_runner.py
- src/ibkr_tax/services/tax_aggregator.py
- src/ibkr_tax/services/excel_export.py
- src/app.py
