# Plan 20.1 Summary: Native App Wrapper

Implemented a native windowing system using `pywebview` and `PySide6`.

## Changes
- Installed `pywebview`, `PySide6`, `bottle`, and `proxy_tools` in the virtual environment.
- Created `src/ibkr_tax/launcher.py` which:
  - Dynamically finds a free port.
  - Launches Streamlit headlessly as a background process (`CREATE_NO_WINDOW` on Windows).
  - Opens a native webview window.
  - Automatically kills the Streamlit server when the window is closed.
